# Design Pattern Exercises (Java)

Each pattern exercise is implemented as a separate folder containing `src/` with Java classes.

## Projects
- SingletonPatternExample
- FactoryMethodPatternExample
- BuilderPatternExample
- AdapterPatternExample
- DecoratorPatternExample
- ProxyPatternExample
- ObserverPatternExample
- StrategyPatternExample
- CommandPatternExample
- MVCPatternExample
- DependencyInjectionExample

## Compile (example)
Compile a project by running:
- `javac <ProjectName>/src/*.java`

## Run (example)
For tests/main classes (example paths):
- SingletonPatternExample: `java SingletonPatternExample.src.LoggerTest`

Notes:
- Some terminals/windows shells may not expand `*.java` reliably. If needed, compile by listing files directly.
