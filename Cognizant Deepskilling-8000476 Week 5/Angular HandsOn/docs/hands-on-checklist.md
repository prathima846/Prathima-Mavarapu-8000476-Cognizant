# Hands-on coverage

1. Environment and project structure: `student-course-portal` with standalone bootstrap, router, styles, and notes in this repository.
2. Binding, lifecycle, input/output: `HomeComponent`, `CourseListComponent`, `CourseCardComponent`.
3. Directives and pipes: structural/attribute directives are demonstrated in course components; extend with `HighlightDirective` and `CreditLabelPipe` as the next exercise step.
4–5. Forms and validation: `EnrollmentComponent` has template-driven bindings and validators; extend it with the requested reactive form variant.
6–7. Services, routing, guards: `CourseService` and parameterized routes are included; add lazy loading and guards during the relevant hands-on.
8. HTTP: `provideHttpClient()` is registered; replace the in-memory service data with JSON Server calls and interceptors.
9. NgRx: install NgRx packages and add the requested store feature files.
10. Tests: add Angular CLI-generated `.spec.ts` files and run `npm.cmd test` after dependencies are installed.
