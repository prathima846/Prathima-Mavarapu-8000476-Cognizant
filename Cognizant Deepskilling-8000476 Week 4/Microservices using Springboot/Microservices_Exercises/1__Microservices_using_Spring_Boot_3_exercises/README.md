# Microservices using Spring Boot 3.0 Exercises

This folder contains exercises for building microservices using Spring Boot 3.0.

## Exercises Included:

### 1. Build a User and Order Management System
- Create User Service microservice
- Create Order Service microservice
- Use REST APIs for inter-service communication
- Use WebClient (Spring WebFlux) or OpenFeign for service communication
- Store data in MySQL or PostgreSQL

### 2. Inventory Management System with Service Discovery
- Product Service: Manage products and stock
- Inventory Service: Track stock levels
- Spring Cloud Netflix Eureka for service discovery
- Spring Cloud Config Server for centralized configuration

### 3. Implement an API Gateway
- API Gateway to route requests to:
  - Customer Service
  - Billing Service
- Spring Cloud Gateway implementation
- Rate limiting, caching, and path rewriting

### 4. Resilient Microservices with Circuit Breaker
- Payment Service calling third-party API
- Circuit Breaker implementation using Resilience4j
- Fallback logic and monitoring

## How to Use:
1. Review the PDF document for detailed exercise descriptions
2. Follow the step-by-step instructions provided
3. Implement each exercise using Spring Boot 3.0
4. Test your microservices thoroughly
